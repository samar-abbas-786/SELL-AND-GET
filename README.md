
# 📚UNIUNITY
  
Welcome to the Hostel & Campus Marketplace website repository! This project aims to provide a platform for students to rent, buy, or sell their products within the hostel and campus community. Whether you're looking for textbooks, furniture, electronics, or even looking to rent out your place, this website facilitates easy transactions among students.





## Features

**User Authentication:** Secure user authentication system for students to create accounts and log in.
**Product Listings:** Students can list their items for rent, sale, or exchange with detailed descriptions and images.

**Search and Filter:** Easy search and filter options to help users find the products they're looking for quickly.

**Messaging System:** Integrated messaging system for users to communicate with each other regarding product inquiries, negotiations, and transactions.

**Transaction Management:** Manage transactions securely within the platform, including payment processing and order tracking.

**Feedback and Ratings:** Rate and review transactions to build trust and transparency within the community.

**Responsive Design:** Mobile-friendly and responsive design to ensure accessibility across devices.

 
 ## Tech Stack 
**Client:** EJS , CSS

**Server:** Node, Express

**Database:** MongoDB




# Installation

### 1.Clone the repository to your local machine:
#### git clone  https://github.com/samar-abbas-786/UniUnity.git

### 2.Install dependencies:
#### (a) cd UniUnity
#### (b) npm install

### 3.Set up environment variables:

- #### Create a .env file in the root directory 
- #### Add necessary environment variables such as database connection URI, JWT secret, etc.
 
 ### 4.Run the server:
   ####  npm start

***Access the website in your browser at http://localhost:3000.***
